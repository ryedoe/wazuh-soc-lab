# Wazuh SOC Lab – Blue Team

Laboratório prático de SOC utilizando Wazuh para detecção, resposta e automação
de incidentes em ambientes Linux e Windows.

## Arquitetura
- Wazuh Manager + Dashboard (Linux)
- Wazuh Agent (Windows e Linux)

## Casos simulados
- Criação não autorizada de usuário
- Tentativa de força bruta
- Alteração crítica de arquivos (FIM)

## Técnicas
- Regras customizadas
- Active Response
- MITRE ATT&CK
- Dashboards SOC e Auditoria

## Objetivo
Demonstrar habilidades práticas de SOC / Blue Team.
